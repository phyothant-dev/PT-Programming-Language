let page = readFile("index.html");

httpListen(3000, (req) => {
  if (req.path is "/") {
    return page;
  }
  return "404";
});

Why PT?

🚀

Unique Syntax

No more print(). Use show(). No ==, use is. PT has its own identity.

🌐

Built-in Server

One line to start a web server. httpListen(3000, handler) and you're live.

📁

File I/O

Read and write files with readFile() and writeFile(). Serve static assets.

🎯

Pattern Matching

Powerful match expressions with => syntax for clean, readable code.

💡

Classes & OOP

Full OOP with classes, inheritance, static methods, this and super.

⚡

Zero Setup

Single binary. No packages, no npm, no build step. Just ./pt server.pt.
